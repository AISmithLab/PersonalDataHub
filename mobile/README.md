# PersonalDataHub — Mobile App

React Native app for Android and iOS.

For build instructions, prerequisites, and troubleshooting see the [Android Build Guide](../systemdesigns/ANDROID-BUILD-GUIDE.md).

## Quick start

```bash
# Install dependencies (from this directory)
npm install

# Build and run on Android
npm run build:android

# Build and run on iOS
npm run build:ios
```

See the build guide for device setup, environment variables, and release builds.
